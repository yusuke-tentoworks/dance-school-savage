クラス - CLASS

クラスや対象はあくまで目安なので、体験などをとおして一人一人に最適なクラスを提案します

KIDS Class (入門)

水曜日 17:00-17:45 土曜日 13:15-14:00
月4回 対象: 年少〜小2

ダンスを全くやったことのない初心者の子にオススメ。ストレッチをして音に合わせて楽しくリズムを取る練習やリズムジャンプ、アイソレーションを主にやっています！

JUNIOR Class (初級)

水曜日 18:00-19:00 (4/22開講) 土曜日 15:30-16:30 (4/25開講)
月4回 対象: 小1〜小3

基礎をしっかり身につけながら、楽しくレベルアップしていくクラスです。ストレッチ・体づくりに加え、リズムトレーニングや基礎ステップ、アイソレーションをバランスよく行います。振付にもチャレンジしながら、表現力やダンスの楽しさを伸ばしていきます。

Girls Style (初級)

土曜日 14:15-15:15
月4回 対象: 小4〜大人

ストレッチをして踊れる身体作りとしてトレーニングをしてから、アイソレーション、リズム取り、振り付けメインのクラスとなってます。ガールズヒップホップやkpopを主に使っていきます。かっこよくて可愛いガールズダンスをやりたい方にオススメ！

MOMS Class

第2第4水曜日 10:00-11:00 (4/22開講)
月2回

運動不足解消やリフレッシュにぴったりのクラス。ストレッチや簡単なリズムトレーニングからスタートし、無理なく体を動かしていきます。ダンスが初めての方でも安心して参加でき、お子様連れもOKなのでお気軽に通っていただけます。お子様連れでも気にならないladyの方ももちろん大歓迎です！

豊橋市在住 Yさん

できない動きがあってもできるまで細かくしっかり優しく教えてくれるので、毎週楽しくレッスン受けてます。

豊川市在住 Mさん

先生たちは明るくてラフで話しやすく、子供達もいつも笑っていて、出来なかったことができるようになる娘を見るたびに親としても嬉しくなります。

子供のやる気がない日でも、気分の上がる曲で先生2人が上手く子供のやる気を出してくれて、すぐに子供も楽しく踊れてます。1人1人の苦手な動きもしっかり教えてくれます。
